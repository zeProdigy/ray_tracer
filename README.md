# ray_tracer
Простой трассировщик лучей на Rust

## Установка Rust и Cargo
`curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`

## Сборка и запуск
`cargo build`

`./target/debug/ray_tracer`
